## PiPup Overlay

**PiPup Overlay** enables Control4 integration with **PiPup**, a picture-in-picture popup application for Android TV.

This driver allows Control4 systems to display **notifications, images, videos, and web pages** as overlay popups directly on the TV screen. The overlays appear in a non-intrusive picture-in-picture window, allowing users to receive alerts and information without interrupting the current content.

### Use Cases

- Doorbell camera popups
- Motion detection alerts
- Security notifications
- Package delivery alerts
- Custom Control4 system messages
- Display images, videos, or web pages on demand
